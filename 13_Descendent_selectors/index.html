Sibling Convinator

All elements inside a ul, form or any other kind of element collector are siblings. Nevertheless, general sibling convintor, which uses ~ operator only allows us to select the direct elements after the element we are targeting.

Adjacent Sibling Convinator

We can see that adjacent convinator uses the + operator which allows us to select one element that comes directly after.
