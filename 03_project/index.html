Here are some insights of html

a and select element are both phrasing content:

  • 'a' non interactive float content
  • 'select' can only contain option and option group elements
  • h1 to h6 heading elements can only content phrasing content.

Specificly designed to contain elements: Nesting elements

section element

  • Contains thematically related elements
  • The section element is part of a whole
  • Here's an example of section: In the homepage we have some merch items and some blog posts. We want to distinguish them like:

article element

  • Self-contained composition (makes sense on its own)
  • The following example from MDN shows a forecast widget

div element

  • Very generic, so useful for grouping
  • Mainly used to aid in styling a page

header element

  • Introductory content
  • Header bars with logo and navigation
  • You can use header inside section and article, to put the title and other information

nav element

  • Major navigation blocks in the site
  • Often inside the header (Particularly when used as a header bar)

main element

  • The main content of the document (what is unique to this page)
  • Only 1 per page, inside the body
  • If we want to provide support fo older browser we should place the special role attribute
  • Ensures that the main element is recognised as the main element content for accessibility

footer element

  • Social media links
  • Secondary navigation
  • Site small print
  • We can use footer inside articles or sections, but they should be related to the parent element
