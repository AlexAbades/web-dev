How to put a class attributes to an html attribute:

  1. After the attribute we place a space and we type class=""
  2. In between the quotation marks we type whatever name we want
  3. class is a global attribute, so it can be applied to anything.
  4. We can put more than one class name inside a class in order to group elements.
    • class="class-one class-two"

We can use instead id attribute. The main difference is that the id should be unique.
