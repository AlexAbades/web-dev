What is BEM?

It stands for Block Element Modifier

Block

Block: Meaningful standalone entity

  • Header
  • Menu bar
  • Form

They should mean something on its own.

Block names are usually single words, but we can also use a hyphen - to join words

Elements

Elements are Part of a block. Has a meaning that is tight to the block.

  • Heade Logo: header__logo
  • Menu Item: menu__item
  • Form Fields: form__fields

Elements are typically separated with a doble underscore or dunderscore

Modifiers

A variant of the basic styles for a block or an element. Allows to no have to crete different ellement names for the same thing

Uses a block element class and a double hyphen to separate the modifying style
